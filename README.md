# 📊 Dashboard Comercial - Performance de Vendas

Projeto desenvolvido durante o curso de Power BI da Data Science Academy com foco em análise de dados comerciais e performance de vendas utilizando dashboards interativos no Power BI.

---

# 🎯 Objetivo do Projeto

Este mini-projeto tem como objetivo analisar a performance comercial de uma empresa fictícia através de diferentes perspectivas de vendas, utilizando recursos de Business Intelligence e visualização de dados no Power BI.

O projeto busca transformar dados comerciais em insights estratégicos para apoio à tomada de decisão.

---

# 🛠️ Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- CSV
- Modelagem de Dados
- Visualização de Dados

---

# 📈 Análises Desenvolvidas

O dashboard foi construído para analisar a performance de vendas sob diferentes perspectivas, permitindo identificar padrões, tendências e indicadores importantes para o negócio.

Entre as análises realizadas estão:

- Performance geral de vendas
- Comparação entre categorias e segmentos
- Análise temporal das vendas
- Identificação de tendências comerciais
- Indicadores de faturamento
- Avaliação de métricas de desempenho

---
